post /token

Create an authentication token and a refresh token for the user identified in the Basic Auth header

  • Request
  • Response

Query Parameters

  • grant_type: required(string)

    Type of grant requested (value must be set to "password" for "Resource Owner Password Credentials Grant" flow or to "client_credentials" for "Client Credentials Grant" flow as per RFC 6749; REQUIRED)

  • username: (string)

    The PRIP username (as per RFC 6749; REQUIRED for "Resource Owner Password Credentials Grant" flow, must not be set otherwise)

  • password: (string)

    The PRIP password (as per RFC 6749; REQUIRED for "Resource Owner Password Credentials Grant" flow, must not be set otherwise)

  • scope: (string)

    The scope of the access request (as per RFC 6749; OPTIONAL, will be ignored if set)

HTTP status code 200

Authentication succeeded, and the token was successfully generated

Body

Media type: application/json

Type: object

Properties
  • access_token: required(string)

    The access token issued by the authorization server (as per RFC 6749)

  • token_type: required(string)

    The type of the token issued (as per RFC 6749; always "bearer")

  • expires_in: required(integer)

    The lifetime in seconds of the access token. For example, the value 3600 denotes that the access token will expire in one hour from the time the response was generated (as per RFC 6749)

Example:

{
	"access_token": "2YotnFZFEjr1zCsicMWpAA",
	"token_type": "bearer",
	"expires_in": 3600
}

HTTP status code 400

Authentication failed

Body

Media type: application/json

Type: object

Properties
  • error: required(string)

    A single ASCII [USASCII] error code from the following: invalid_request, unauthorized_client, unsupported_grant_type (subset of values defined in RFC 6749)

  • error_description: required(string)

    Human-readable ASCII [USASCII] text providing additional information (as per RFC 6749)

Example:

{
	"error": "unauthorized_client",
	"error_description": "(E5008) User S5P\\pripuser not authorized for PRIP API"
}
