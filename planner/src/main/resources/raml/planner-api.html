Get production planner jobs by state

  • Request
  • Response

Body

Media type: application/json

Type: array of object

Items: PlannerJob

  • id: required(string)
  • description: (string)

Example:

  [
    { "id" : "0ae234-887f26", "description" : "Produce S5P.TROPOMI.CO2 products for orbits 2345-4567" },
